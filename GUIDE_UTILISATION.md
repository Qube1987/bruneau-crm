# Guide d'Utilisation - CRM Bruneau Protection

## Connexion

1. Ouvrez l'application
2. Entrez vos identifiants
3. Cliquez sur "Se connecter"

---

## Créer une Nouvelle Opportunité

### Méthode 1 : Depuis un Client Extrabat

1. **Rechercher le client**
   - Tapez le nom du client dans la barre de recherche
   - Cliquez sur "Rechercher dans Extrabat"
   - Sélectionnez le bon client dans les résultats

2. **Compléter les informations**
   - Un modal s'ouvre automatiquement
   - Renseignez la description de l'opportunité
   - Vérifiez/modifiez le statut si nécessaire
   - Indiquez le montant estimé et la date des travaux

3. **Enregistrer**
   - Cliquez sur "Enregistrer"
   - **Le SMS de notification est envoyé à ce moment**

### Méthode 2 : Créer un Nouveau Client

1. **Ouvrir le formulaire**
   - Cliquez sur "Nouveau Client"

2. **Remplir les informations**
   - Civilité et identité
   - Téléphone et email
   - Adresse complète
   - Origine du contact
   - Qui assure le suivi

3. **Valider**
   - Cliquez sur "Créer le Client"
   - Le client est créé dans Extrabat et dans le CRM
   - Un modal s'ouvre pour compléter l'opportunité
   - **Renseignez les détails et cliquez sur "Enregistrer" pour envoyer le SMS**

---

## Gérer les Opportunités

### Changer le Statut

Les statuts disponibles :
- **À contacter** (rouge) : Client à rappeler
- **En cours** (bleu) : Négociation en cours
- **Devis envoyé** (violet) : Devis transmis au client
- **Standby** (orange) : En attente
- **Gagné** (vert) : Affaire conclue
- **Perdu** (gris) : Affaire perdue

Pour changer le statut :
1. Cliquez sur le statut actuel de l'opportunité
2. Sélectionnez le nouveau statut dans la liste

### Modifier une Opportunité

1. Cliquez sur l'opportunité
2. Modifiez les informations nécessaires
3. Cliquez sur "Enregistrer"

### Marquer comme Prioritaire

1. Cliquez sur l'étoile à côté de l'opportunité
2. Les opportunités prioritaires apparaissent en haut de la liste

---

## Ajouter des Interactions

Les interactions permettent de garder un historique des échanges avec le client.

1. **Ouvrir l'opportunité** en cliquant dessus
2. **Cliquer sur "Ajouter une interaction"**
3. **Choisir le type** :
   - Téléphonique
   - Email
   - Visite
   - Visioconférence
   - Autre
4. **Rédiger la note** avec les détails de l'échange
5. **Enregistrer**

---

## Envoyer au Service Devis

Quand une opportunité est prête à être transformée en devis :

1. **Ouvrir l'opportunité**
2. **Cliquer sur "Envoyer au Devis"**
3. **Vérifier les informations** :
   - Client
   - Description
   - Montant estimé
   - Date des travaux
4. **Valider l'envoi**
5. L'opportunité passe automatiquement en statut "Devis envoyé"

## Filtres et Recherche

### Filtrer par Statut
- Cliquez sur les onglets en haut pour filtrer les opportunités par statut

### Filtrer par Commercial
- Utilisez le sélecteur "Suivi par" pour voir uniquement vos opportunités

### Rechercher un Client
- Tapez dans la barre de recherche
- Cherchez dans Extrabat pour importer de nouveaux clients

---

## Actions Commerciales

### Ajouter une Action Planifiée

1. Allez dans l'onglet "Actions commerciales"
2. Cliquez sur "Nouvelle Action"
3. Renseignez :
   - Type d'action
   - Description
   - Date prévue
   - Responsable
4. Enregistrez

### Suivre les Actions

Les actions sont triées par date. Les actions passées apparaissent en rouge.

## Conseils d'Utilisation

### Bonnes Pratiques

✅ **Ajouter des interactions régulièrement** pour garder un historique complet

✅ **Mettre à jour les statuts** dès qu'une opportunité évolue

✅ **Utiliser les descriptions** pour noter les détails importants

✅ **Marquer comme prioritaire** les affaires urgentes

✅ **Renseigner les montants** pour un meilleur suivi commercial

### Ce qui Déclenche le SMS

⚠️ **Le SMS de notification est envoyé uniquement quand :**
- Vous créez une nouvelle opportunité depuis Extrabat
- Vous complétez les informations dans le modal
- **Vous cliquez sur "Enregistrer"**

Le SMS contient :
- Le nom du client
- La description de l'opportunité
